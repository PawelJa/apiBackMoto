# apiBackMoto

Front-end and back-end application to managment events.

Functionality:
  - registry and login panel
  - safe login panel
  - form to add events and feedbacks
  - possibility to join to event and add user avatar

Possible extensions:
  - saving event gallery on cloud
  - users rules

Used technologies:
  - Java
  - Spring
  - Spring Security
  - Hibernate
  - JavaScript
  - MySql
  - HTML
  - CSS
  
IDE:
  - IntelliJ
